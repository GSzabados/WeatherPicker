    -------------------------
    Simple weather information including a 7-day forecast for the clicked location. Temperature and rainfall are rendered as a chart. Data is provided by the Open-Meteo API (https://open-meteo.com). Network requests go through the QGIS network manager and therefore honour the proxy settings configured in QGIS.
